# HW05（課題05）

答えだけを書くのではなく，おおよその解法を説明すること。方針だけでもよい。

````{admonition} 問1
:class: exercise

関数 $f$ を $f(x)=x^3-x+1$ と定める。

1. 関数 $y=f(x)$ のグラフの概形を描け。
2. ブラケット $[a,b]$ の初期値を適当に定め，二分法を用いて方程式 $f(x)=0$ の解を小数点以下3桁まで求めよ。
````

````{admonition} 問2
:class: exercise

関数 $f$ を $f(x)=\cos(x)-x$ と定める。

1. 関数 $y=f(x)$ のグラフの概形を描け。
2. ブラケット $[a,b]$ の初期値を適当に定め，二分法を用いて方程式 $f(x)=0$ の解を小数点以下3桁まで求めよ。
````
